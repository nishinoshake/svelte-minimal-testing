# svelte-minimal-testing

## Tools
* [Jest](https://jestjs.io)
* [Cypress](https://www.cypress.io)
* [@testing-library/svelte](https://github.com/testing-library/svelte-testing-library)

## Commands
```
npm install

npm run dev

npm run build

npm run test:unit

# run after build
npm run test:e2e
```
